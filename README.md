# 🦉 OlwSchool

**OlwSchool** é um mini sistema acadêmico desenvolvido como projeto da disciplina de Engenharia de Software (PUCPR – Curitiba).  
O objetivo é oferecer uma plataforma simples para que **pais, alunos, professores e gestores** acompanhem a vida escolar em um único portal.

---

## 🚀 Funcionalidades (MVP)
- Login para **aluno, professor e responsável**  
- Consulta de **notas e frequência**  
- Cadastro de tarefas e notificações para os responsáveis  
- Área financeira (mensalidades)  
- Visualização de cursos, turmas e disciplinas  

---

## 🛠️ Tecnologias
- **Frontend:** HTML + CSS (Bootstrap)  
- **Backend:** Python (Flask + Jinja2)  
- **Banco de Dados:** MySQL  
- **Controle de versão:** Git/GitHub  

---
